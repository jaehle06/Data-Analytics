# upload jupyter notebook on GitHub
1. 로컬 저장소 만들기 
  - git init 
2. 원격 저장소 불러오기
  - github 에서 원격저장소 만든다음 로컬과 연결
  - git clone 사용자명@호스트:~(clone or download 버튼 누르면 주소 확인가능)
3. 작업 내용 업로드
  - git과 원격 저장소에 반영
  - git add 
  - git commit -m (확정본에 대한 설명 추가해서 커밋)
  - 만약 원격 저장소와 로컬 저장소 폴터 구성 다르면 git pull로 둘의 상태 같게 만들기
